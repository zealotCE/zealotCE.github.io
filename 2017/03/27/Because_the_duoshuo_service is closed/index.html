因为多说评论服务关闭

发表于 2017-03-27 | 分类于 hexo | 阅读次数

在3月22日的晚上，一个普通的周末突然看到了一封意外的邮件。重要通知：多说即将下线

当时只有震惊与难过了。国内的网络问题导致Disqus基本用不了，而多说这一倒突然有点儿难以适从，毕竟这评论和分享得重新弄了。或许：免费的才是最贵的

刚刚把博客的评论和分享分别替换成了来必力和Add this

来必力评论

网易云跟帖和搜狐畅言有些不太好的毛病导致我没有使用。

来必力是一个韩国人开发的评论系统，网站有种文化但是不完全，在登陆验证那里还是韩文。不过还可以用吧。

使用方法：登陆/注册来必力，获取你的 LiveRe UID（来比力网站提供的代码中的data-uid字段的值），填写到next主题配置文件中

Add this分享

Add this是一个比较优秀的分享服务，支持国内多个网站且分享的样式等等都可定制挺好用的。

Add this可以直接使用Google+或Facebook直接登陆。定制好你的分享后从下面菜单获得 AddThis id:More.. –> General –> ID

好像一切都没变，但是一切都变了。
