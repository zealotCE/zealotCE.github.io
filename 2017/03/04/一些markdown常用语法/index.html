一些 Markdown 常用语法

以前用Word写东西，总是要拿鼠标各种修改格式什么的真的好麻烦，直到我遇到了Markdown。

其实知道Markdown已经很久了，但是各大网站大多都不支持知乎就是你（简书什么的没听说过）。正好某天看到了hexo这个支持Markdown语法的博客系统，终于满足了我用Markdown乱写东西的心（逃

后面还有很多，慢慢补充

删除线

只要加上两个~号,左右包裹住内容就可以了。

~~删除线~~

标识

像这样

>标识

斜体

只要*号或者_号包裹住就好了

比如 *斜体* 或者斜体

加粗

两个*号或者_号包裹住

比如： *加粗的内容*

倾斜加粗

三个*号或者_号包裹住

比如：倾斜加粗 ***倾斜加粗*

无序列表

在每行开头加 + / - / * 号都可以，记得加一个空格。

- zealotCE的博客

+ zealotCE的博客

* zealotCE的博客

效果都是:

  • zealotCE的博客

有序列表

在每行开头加上序号和英文句号就好了，记得加空格。
